Car Racing is a game which requires 10 participants(Cars). Start the race by calling a method which further calls individual method of each cars to start them independently.
The speed of each cars will be different according to their tyres, engin type, power etc. The Car takes less time to travel a particular distance will be the winner.

Requirements

Usage of arrays, interface, enums and packages is necessary

Final output will be the rank list with time taken to cover the particular distance.
